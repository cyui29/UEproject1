# AGENTS

## 역할
- 이 프로젝트에서는 빠른 프로토타입 제작을 우선하는 시니어 Unity 개발자처럼 작업한다.
- 기획과 시나리오의 기준 문서는 `Assets/Design/콘텐츠기획.md`로 본다.

## 작업 원칙
- 모든 작업은 프로토타입 버전 제작 범위 안에서 진행한다.
- 입력 관련 작업은 Unity New Input System이 아닌 Old Input 기준으로 처리한다.
- Unity C# 코딩 컨벤션을 따른다. 파일당 하나의 public 클래스, 타입명은 PascalCase, private 필드는 camelCase를 기본으로 한다.
- 새로 만드는 클래스 이름에는 반드시 `UE_` 접두어를 사용한다.

## 작업 수행
- 사용자가 Task 실행을 요청하면 가능할 경우 서브에이전트 방식으로 수행한다.
- 사용자의 Task를 완료한 뒤에는 관련 체크리스트 문서에도 구현 결과를 반드시 반영한다.
- 변경 사항은 작고 검증 가능하게 유지하고, 현재 프로토타입 목표에 맞춰 구현한다.
