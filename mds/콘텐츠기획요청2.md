# 역할
너는 20년차 unity 콘텐츠 기획자야.

## 배경
window pc의 카메라를 이용해서 내 손을 트래킹하여 인터렉션하는 콘텐츠를 만드는 프로젝트를 진행하려고 해. 3D 가상공간에서 탈출하는 게임이야. 나의 왼손으로 이동하고 오른손으로 인터랙션을 해서 각 단계를 깨서 방을 탈출해야돼. 
- 실행 환경: PC windows

## 요구사항
이 프로젝트를 만들기 위해 시나리오 기획, 콘텐츠 기획서를 작성하려고 해. 우선 핸드트래킹은 Mediapipe를 이용할거고 이미 패키지는 import 해놨어. 
`Assets/Design` 폴더의 `콘텐츠기획2.md`에 다음 내용을 작성해줘.

1. 시나리오 기획:
- 컨셉 및 기획 의도: 핵심 메시지와 목적 정의
- 타겟 정의: 누구를 위한 컨텐츠인지 분석
- 사용자 인터렉션: 사용자의 손짓을 어떻게 하면 어떤 식으로 인터랙션이 되는지
    - 예시) 왼손을 어떻게 해야 이동, 멈춤, 점프
            오른손 핀치로 오브젝트 이동, 주먹쥐기로 오브젝트 부시기
        

2. 콘텐츠 기획:
- 작품 라인업(스펙): 필요한 기술, SDK, 이미지, 영상, 오디오 등의 목록 및 메타데이터
- 공간 테마 및 배치: 섹션별 공간 디자인 모드(색상, 조명)와 작품 배치 위치
    - 각각의 단계를 수행할 수 있는 방구조
- 스토리텔링 및 텍스트: 작품 설명(도슨트 스크립트), 월 텍스트, 팝업 안내문
- 기술 사양: 지원 플랫폼(Windows, 웹 등)
- 콘셉트 이미지 / 무드보드 
    - 역할: 인터렉션하는 모습과 화면 UI의 전체적인 분위기, 색감, 느낌을 한눈에 보여주는 이미지 모음.
    - 포함 내용: 레퍼런스 사진, 사용할 색상 팔레트, 텍스트 폰트 스타일, 조명 효과 예시 등.
    - 실제 콘셉트 이미지 생성해서 `Assets/Concept` 에 파일 저장

- 이미지들은 svg 말고 png로 생성.
