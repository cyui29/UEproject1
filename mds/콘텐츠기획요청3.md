# 역할
너는 20년차 unity 콘텐츠 기획자야.

## 배경
window pc의 카메라를 이용해서 내 손을 트래킹하여 3D 오브젝트와 인터렉션하는 콘텐츠를 만드는 프로젝트를 진행하려고 해. 내 카메라는 화면에 안보이고 3D 가상공간에서 player가 움직일거야. 나의 왼손으로 이동하고 오른손으로 공간 안의 각 오브젝트와 인터랙션할거야.
- 실행 환경: PC windows

## 요구사항
이 프로젝트를 만들기 위해 시나리오 기획, 콘텐츠 기획서를 작성하려고 해. 우선 핸드트래킹은 Mediapipe를 이용할거고 이미 패키지는 import 해놨어. `Assets/MediaPipe/Samples/Hand Landmark Detection`하위의 씬과 스크립트들을 활용해봐. 
`Assets/Design` 폴더의 `콘텐츠기획3.md`에 다음 내용을 작성해줘.

1. 시나리오 기획:
- 컨셉 및 기획 의도: 핵심 메시지와 목적 정의
- 타겟 정의: 누구를 위한 컨텐츠인지 분석
- 사용자 인터렉션: 사용자의 손짓을 어떻게 하면 어떤 식으로 인터랙션이 되는지
    - 왼손: 이동
        예시) 화면을 반으로 가르고 펼친 왼손을 원 안에서 왼쪽으로가면 왼쪽으로 이동, 오른쪽으로가면 오른쪽으로 이동, 위로 가면 카메라가 위쪽을 응시, 아래로 가면 아래를 응시, 주먹을 쥐면 멈춤
    - 오른손: 오브젝트와 인터랙션
        예시) 가운데 있는 타겟 십자가(?)가 오브젝트를 타겟하고 오른손을 핀치하면 오브젝트 집기(오브젝트가 1m앞으로 옴), 오른손 주먹쥐기로 오브젝트 부시기 등
        

2. 콘텐츠 기획:
- 작품 라인업(스펙): 필요한 기술, 이미지, 영상, 오디오 등의 목록 및 메타데이터
- 공간 테마 및 배치: 귀엽게 꾸며진 공간(한개의 방)과 인터랙션 할 수 있는 오브젝트
    - 하나의 공간
- 스토리텔링 및 텍스트: 인터랙션 설명(오브젝트 근처로 가면 인터랙션 가이드 자막이 뜸)
- 기술 사양: 지원 플랫폼(Windows, 웹 등)
- 콘셉트 이미지 / 무드보드 
    - 역할: 인터렉션하는 모습과 화면 UI의 전체적인 분위기, 색감, 느낌을 한눈에 보여주는 이미지 모음.
    - 포함 내용: 3D공간의 2D평면도, 레퍼런스 사진, 사용할 색상 팔레트, 텍스트 폰트 스타일, 조명 효과 예시 등.
    - 실제 콘셉트 이미지 생성해서 `Assets/Concept` 에 파일 저장

- 이미지들은 svg 말고 png로 생성.
