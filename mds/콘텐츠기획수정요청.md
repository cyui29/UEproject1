# 역할
너는 20년차 unity 콘텐츠 기획자야.

## 배경
현재 mediapipe 패키지를 import 한 상태야. 나는 일단 Hand detection 부분만 사용할거야. 그런데 나는 touchdesigner라는 tool을 알게됐는데, 그거랑 좀 차별화한 기획을 하고싶어. 
- 실행 환경: PC windows

## 요구사항
- unity에서만 할수있는, 혹은 더 다루기 쉬운, 혹은 퀄리티가 더 좋게 만들 수 있는 부분을 기획해줘.
- `Design/콘텐츠기획.md`를 보고 위의 부분을 반영해서 수정하고 '콘텐츠기획v2.md'를 만들어줘.
